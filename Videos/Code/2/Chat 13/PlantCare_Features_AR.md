# PlantCare — قائمة ميزات التطبيق

> **الغرض:** شرح ميزات التطبيق بلغة المستخدم — ماذا يستطيع أن يفعل، وماذا يحدث له فعلياً.
> **آخر تحديث:** 2026-05-05
> **عدد الميزات:** ~140 ميزة موزّعة على 23 مجالاً وظيفياً.

كل ميزة لها معرّف فريد (مثل F5.1) للتسهيل عند الإشارة إليها.

---

## 1. الدخول والحساب

| المعرّف | الميزة | ماذا تعني للمستخدم |
|---------|---------|------------------------|
| F1.1 | شاشات الترحيب | عند فتح التطبيق لأوّل مرّة، يستقبلك ٤ شاشات تشرح ميزاته الأساسية وتطلب موافقتك على شروط حماية البيانات (DSGVO). |
| F1.2 | الدخول كضيف | يمكنك تجربة التطبيق دون إنشاء حساب أو إدخال بريد. كل بياناتك تُحفَظ على جهازك فقط. |
| F1.3 | تسجيل ببريد وكلمة سر | إنشاء حساب جديد ببريدك. كلمة المرور تُشفَّر تلقائياً قبل الحفظ. |
| F1.4 | تسجيل دخول بالبريد | الدخول لحسابك الموجود من نفس الجهاز أو من جهاز جديد. |
| F1.5 | تسجيل دخول بحساب Google | دخول بنقرة واحدة عبر Google بدون كلمة مرور. |
| F1.6 | فتح حوار الدخول من أيّ مكان | إذا كنت ضيفاً وحاولت ميزة تتطلّب حساباً، يفتح حوار الدخول مباشرة. |
| F1.7 | ترقية تشفير كلمة السر تلقائياً | المستخدمون القدامى الذين سجّلوا قبل التحديث، تُحدَّث طريقة تشفير كلماتهم بصمت دون إزعاج. |
| F1.8 | تغيير كلمة المرور | من الإعدادات. |
| F1.9 | تعديل اسم العرض | يظهر اسمك الجديد في كل المكان مباشرة. |
| F1.10 | حذف الحساب | يحذف كل نباتاتك وتذكيراتك وصورك من الجهاز والسحابة. لا رجوع. |
| F1.11 | تسجيل الخروج | يخرجك من حسابك مع الاحتفاظ بإمكانية الدخول مجدّداً. |

## 2. إدارة النباتات

| المعرّف | الميزة | ماذا تعني للمستخدم |
|---------|---------|------------------------|
| F2.1 | إضافة نبتة من الكاتالوج | تختار نبتة من قائمة 506 نبتة جاهزة، تُملأ بياناتها تلقائياً (إضاءة، ري، تربة). |
| F2.2 | إضافة سريعة بضغطة | من قائمة الكاتالوج، اضغط مرّة على نبتة، تُضاف فوراً لـ"نباتاتي" بإعدادات افتراضية. |
| F2.3 | إضافة نبتة مخصّصة | لو نبتتك ليست في الكاتالوج، أنشئها من الصفر بمعلوماتك الخاصة. |
| F2.4 | تعديل نبتة موجودة | تغيير معلومات الرعاية أو ملاحظاتك الشخصية. |
| F2.5 | حذف نبتة | يحذف النبتة وكل تذكيراتها وصورها معها. |
| F2.6 | نقل النبتة لغرفة أخرى | في حال أعدت ترتيب منزلك. |
| F2.7 | إعطاء النبتة اسم مستعار | لتمييز نبتتين من نفس النوع (مثل "صبّار المطبخ" و "صبّار البلكونة"). |
| F2.8 | ملاحظات شخصية | نصّ حر لكل نبتة — مكان شراء، ذكرى، توصيات صديق. |
| F2.9 | إضافة نبتة للمفضّلة | بنجمة، تظهر في الأعلى. |
| F2.10 | حدّ ٨ نباتات للحساب المجاني | بعد ٨ نباتات، يفتح عرض الترقية لحساب Pro. |

## 5. تذكيرات السقاية

| المعرّف | الميزة | ماذا تعني للمستخدم |
|---------|---------|------------------------|
| F5.1 | إنشاء تذكير تلقائي | بمجرّد إضافة نبتة، يُولَّد جدول سقاية كامل لها (٥ / ٧ / ١٤ / ٢١ يوم حسب نوعها). |
| F5.2 | تذكير يدوي | لو أردت تذكير لشيء آخر (تسميد، تقليم)، أنشئه بتاريخ مخصّص. |
| F5.3 | تكرار التذكير | يومياً، كل N يوم، أو مرّة واحدة. |
| F5.4 | تعديل تذكير | تغيير التاريخ أو نمط التكرار. |
| F5.5 | حذف تذكير | فردي أو السلسلة كلّها. |
| F5.6 | تأشير "تمّت" | بنقرة على المربّع، ينتقل لقائمة المنجزات. |
| F5.7 | تأشير الكل دفعة واحدة | في يوم نشط، اضغط مرّة لتسجيل ري كل اليوم. |
| F5.8 | من سقى النبتة | في حسابات العائلة، يُسجَّل من سقى أيّ نبتة. |
| F5.9 | ملاحظة على السقاية | "بدت ظمآنة" أو "أوراقها صفراء" — حقل نصّ حر. |
| F5.10 | إعادة جدولة من اليوم | لو فاتك التذكير، اضغط زر يعيد جدولة التذكير ابتداء من اليوم. |
| F5.11 | قائمة اليوم والمتأخّر | تبويب "اليوم" يُظهر تذكيرات اليوم + ما فات. |
| F5.12 | تذكيرات خطّة العلاج | عند تشخيص مرض، تُنشأ ٤ تذكيرات علاجية تلقائياً. |

## 8. التعرّف على النبتة

| F8.2 | استخدام خدمة PlantNet | قاعدة بيانات نباتية علمية ضخمة. |
| F8.3 | أعلى ٣ احتمالات | بدلاً من خيار واحد، يعرض ٣ احتمالات بنسبة الثقة. |

| F8.9 | تحديد جدول السقاية | بناء على نوع النبتة (صبّار = ٢١ يوم، استوائية = ٧ أيام). |

## 9. تشخيص الأمراض

| المعرّف | الميزة | ماذا تعني للمستخدم |

| F9.7 | سجلّ التشخيصات | تاريخ كامل لكل فحوصاتك. |

| F9.11 | إذن قبل الرفع السحابي | حوار يُعلِمك أنّ الصورة سترسَل لـ Google قبل التشخيص (التزام DSGVO). |
